# Config Validator
Changes the params of a given validator.

## Payload Parameters

|  Required  |    Parameter        |    Type    |               Description                |
|:----------:|----------------------|------------|------------------------------------------|
|   &#9744;  |         name         |   string   | Change validator name                   |
|   &#9744;  |   rewardAddress      |   string   | Change rewards address                  |
|   &#9744;  |    blsPublickey      |   string   | Change BLS public key                   |
|   &#9744;  |     canDelegate      |  boolean   | Change permission to delegate           |
|   &#9744;  |     commission       |   number   | Change commission (precision 2)         |
|   &#9744;  | maxDelegationAmount  |   number   | Change max delegation amount             |
|   &#9744;  |         logo         |   string   | Change logo                              |
|   &#9744;  |         uris         | {[key: string]: string} | Change uris   |
