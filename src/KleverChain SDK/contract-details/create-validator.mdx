# Create Validator
Creates a validator for an address, controlled by a BLS key.

## Payload Parameters
|  Required  |      Parameter       |    Type    |           Description               |
|:----------:|----------------------|------------|-------------------------------------|
|   &#9744;  |        name          |   string   | Validator name                      |
|   &#9744;  |       address        |   string   | Owner address                       |
|   &#9744;  |    rewardAddress     |   string   | Rewards address                     |
|   &#9744;  |    blsPublickey      |   string   | BLS public key                      |
|   &#9744;  |     canDelegate      |  boolean   | Permission to delegate              |
|   &#9744;  | maxDelegationAmount  |   number   | Max delegation amount               |
|   &#9744;  |     commission       |   number   | Validation commission (precision 2) |
|   &#9744;  |        logo          |   string   | Logo URI                            |
|   &#9744;  |        uris          | {[key: string]: string} | Relevant URI's          |
