# Proposal
Creates a proposal to change the chain network parameters.

## Payload Parameters
|  Required  |     Parameter     |              Type             |               Description                |
|:----------:|-------------------|------------------------------|------------------------------------------|
|   &#9744;  |    parameters     | {[paramKey: number]: string} | A object that includes all chain parameters you would like to propose a change. (The value will most likely be a numeric string for current parameters). See parameter list below for correct key input. |
|   &#9744;  |  epochsDuration   |            number            | Proposal epochs duration                |
|   &#9744;  |   description     |           string             | Proposal description                    |
