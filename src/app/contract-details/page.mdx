# Contract Details

Additional info on contract parameters and use cases

The parameters are used in the [IPayload](undefined) in the contract

E.g.:
```tsx
const payload: ITransfer = {
  amount: 100 * 10 ** 6,
  receiver: "receiverAddress",
  kda: "KLV",
};

const Contract: IContract = {
    type: TransactionType.Transfer,
    payload,
  }
```

## Transfer
Transfer assets to another address.

### Payload Parameters

| Required | Parameter | Type   | Description                     |
|----------|-----------|--------|---------------------------------|
| &#9745;  | receiver  | string | Receiver address                |
| &#9745;  | amount    | number | Amount to be sent (with precision) |
| <input type="checkbox"  disabled/>  | kda       | string | AssetID. If empty, defaults to KLV |


## Freeze
Freezes a certain amount of a token for staking. This frozen amount will be referred as a "bucket".

### Payload Parameters
| Required | Parameter | Type   | Description                     |
|----------|-----------|--------|---------------------------------|
| &#9745;  | amount    | number | Amount to be frozen (with precision) |
| <input type="checkbox"  disabled/>   | kda       | string | Defaults to KLV                  |


## Unfreeze
Unfreezes a bucket.

### Payload Parameters
| Required | Parameter | Type   | Description                   |
|----------|-----------|--------|-------------------------------|
| &#9745;  | bucketld  | string | Bucket to be unfrozen         |
| <input type="checkbox"  disabled/>   | kda       | string | Unfrozen asset ID, defaults to KLV if empty |

## Delegate
Delegates a bucket to a validator.
Buckets can only be delegated to a validator address.

### Payload Parameters
| Required | Parameter | Type   | Description            |
|----------|-----------|--------|------------------------|
| &#9745;  | receiver  | string | Address to be delegated to |
| &#9745;  | bucketld  | string | Bucket to be delegated    |


## Undelegate
Removes the delegation of a bucket.

### Payload Parameters
| Required | Parameter | Type   | Description            |
|----------|-----------|--------|------------------------|
| &#9745;  | bucketld  | string | Bucket ID to be undelegated |
